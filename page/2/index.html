第一章 我要做，我不要，我想要：什么是意志力？为什么意志力至关重要？

牢记“我想要”（我真正想要的东西），我真正想要的不是晚睡晚起，不想要吃好吃的东西，不是想要拖延看动漫不是放纵身心开始摆烂。我真正想要的是一个优秀的毕业设计，一份待遇好的工作，一个健康苗条的身体。

更难的事情

比如去学习做毕设找工作。为什么很困难？因为我更想去看动漫刷微博这些很容易的事情，这些可能是获得短期快感，但是对个人提升没有任何意义，这些东西阻扰了我的成长。但我去浪费时间去刷微博刷动漫后，我会觉得巨大的无力感，这种感觉非常非常的不好。

视频 浏览器是如何运作的

浏览器是个多进程结构

  1. 浏览器进程
    • 负责控制Chrome浏览器除标签页外的用户界面包括地址栏书签后退和前进按钮以及负责与浏览器的其他进程序的工作
  2. 网络进程
    • 负责发起接受网络请求
  3. GPU进程
    • 负责整个浏览器界面的渲染
  4. 插件进程
    • 负责控制网站使用了所有插件例如flash
  5. 缓存进程
  6. 渲染器进程
    • 用来控制显示tab标签的所有内容
  • 回流reflow
    • 当render tree 中的一部分(或全部）因为元素的规模尺寸，布局，隐藏等改变而需要重新构建。这就称为回流
  • 重绘repaint
    • 当render tree 中的一些元素需要更新属性，而这些属性只是影响元素的外观，风格，而不会影响布局的。则叫重绘( repaint )。比如颜色改变。
