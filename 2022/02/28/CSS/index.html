CSS 和 HTML 结合的方式(CSS应该放在哪里比较合适)

  1. 行内样式

    • <p style="color:whitebackground-color:red">...</p>

  2. 内嵌样式表

    • 在<head>标签中加入 <style>

  3. 外部样式表

    1. <link>
      • <link rel = "stylesheet" type = "text/css" href = "a.css"></link>
    2. import
      • <style>@import url(a.css) ;</style>

CSS选择器

CSS基本选择器

  1. 标签选择器
    • p{ font-size:14px; }
  2. ID选择器
    -id="" #
    • 不可出现同样的id
  3. 类选择器
    • class="" .
    • 不要去试图用一个类名，把某个标签的所有样式写完。这个标签要多携带几个类，共同完成这个标签的样式。
    • 每一个类要尽可能小，有“公共”的概念，能够让更多的标签使用。
    • 尽可能的用 class，除非极特殊的情况可以用 id
    • 类上样式，id 上行为

CSS属性

  • CSS的单位
    1. 绝对单位
      • in cm mm pt pc
    2. 相对单位
      • px em %
  • font字体属性
    • line-height:xxpx行高
    • font-size:xxpx字体大小
    • font-family:黑体|宋体
    • font-style:italic|normal
    • fon-weight:bold
    • 连写
      • 是否加粗、字号 font-size、行高 line-height、字体 font-famil
      • 至少要有字号和字体
  • 文本属性
    • 文本属性
  • 列表属性
    • 列表属性
  • overflow属性
    • overflow: visible|hidden|auto|scroll
  • 鼠标的属性 cursor
    • p:hover{cursor: pointer;}
  • 背景属性
    • background-color:#ff99ff;
    • background-image:url(images/2.gif);
    • background-repeat: no-repeat|repeat-x|repeat-y;
    • background-position:
      1. background-position:向右偏移量 向下偏移量;
      2. background-position: 描述左右的词 描述上下的词;
        • 描述左右的词：left、center、right
        • 描述上下的词：top 、center、bottom
    • background-attachment: scroll|fixed
    • CSS3新增
      • background-origin=padding-box|border-box|content-box
      • background-clip 背景裁切

      • background-size: 500px 500px|cover|contain 调整尺寸
