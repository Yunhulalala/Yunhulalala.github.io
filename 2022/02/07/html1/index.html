浏览器工作原理

介绍文章 浏览器的工作原理：新式网络浏览器幕后揭秘

了解从在地址栏输入 google.com 直到在浏览器屏幕上看到 Google 首页的整个过程中都发生了些什么。

  1. 简介
    • 浏览器的主要功能
      向服务器发出请求，在浏览器窗口中展示您选择的网络资源.
    • 浏览器的高层结构
      • 关于呈现引擎(Rendering engine)
        负责显示请求的内容。如果请求的内容是 HTML，它就负责解析 HTML 和 CSS 内容，并将解析后的内容显示在屏幕上。
        WebKit 是一种开放源代码呈现引擎(Safari 和 Chrome 浏览器使用)
      • 呈现引擎的流程

视频 浏览器是如何运作的

浏览器是个多进程结构

  1. 浏览器进程
    • 负责控制Chrome浏览器除标签页外的用户界面包括地址栏书签后退和前进按钮以及负责与浏览器的其他进程序的工作
  2. 网络进程
    • 负责发起接受网络请求
  3. GPU进程
    • 负责整个浏览器界面的渲染
  4. 插件进程
    • 负责控制网站使用了所有插件例如flash
  5. 缓存进程
  6. 渲染器进程
    • 用来控制显示tab标签的所有内容

分等级的HTML标签

  1. 文本级标签：p、span、a、b、i、u、em。文本级标签里只能放文字、图片、表单元素。（a标签里不能放a和input）
  2. 容器级标签：div、h系列、li、dt、dd。容器级标签里可以放置任何东西

水平线标签
<hr />
align = "left\center\right" size=2 color="#"

换行标签
<br />

<div><span>标签
介绍

  • <div>：可以把标签中的内容分割为独立的区块。必须单独占据一行。容器级标签
  • <span>：和div的作用一致，但不换行。文本级标签

内容居中标签
<center>不建议使用

预定义（预格式化）标签
<pre>

字体标签

  • 特殊字符
    • &nbsp
    • &lt &gt 大于号小于号
    • 其他特殊字符
  • 下划线、中划线、斜体
    • <u>：下划线标记
    • <s>或<del>：中划线标记（删除线）
    • <i>或<em>：斜体标记
  • 上标
    • <sup>
  • 下标
    • <sub

超链接

  1. 外部链接

    • <a href="02页面.html">点击进入另外一个文件</a>
    • <a href="http://www.baidu.com" target="_blank">点击进入一个网址</a>

  2. 锚链接

    • 在本页面或者其他页面的的不同位置进行跳转
    • 做法，创建一个锚点（用name、id起个名字）
      • <a name = "name1"></a>
      • <a href = "#name1"></a>

  3. 邮件链接

    • <a href="mailto:xxx@163.com">点击进入我的邮箱</a>

HTML5

介绍

HTML5是作为HTML标记语言的一个最新版本，它制定了Web应用开发的一系列标准，成为第一个将Web做为应用开发平台的HTML语言。
HTML5是新一代开发 Web 具有很强的交互性和体验的应用程序整体解决方案。包括：HTML5，CSS3，Javascript API在内的一套技术组合。
