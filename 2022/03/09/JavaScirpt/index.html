JavaScript 基础

书写方式

  • 外链式
    • 在body里面引入外部的js文件

      变量和常量

  • 常量的定义
    • const 常量名称 = 常量取值;
  • 变量的定义
    • ES5:var name;
    • ES6: let age
  • 变量名中不允许出现空格。尤其注意，变量名中不能出现中划线-

    变量的数据类型

  1. 基本数据类型（值类型）
    • String 字符串、Number 数值、BigInt 大型数值、Boolean 布尔值、Null 空值、Undefined 未定义、Symbol。
    • 字符串
      • 模板字符串
        • console.log(我是${name},age:${age}); //ES6 写法。注意语法格式
        • 插入变量、表达式、调用函数、 可嵌套
    • Number
      • 数值 整数和小数
      • 无穷大（正无穷）：Infinity 无穷小（负无穷）：-Infinity
      • NaN 非数值
      • 隐式转换：- * / %
      • 浮点数的运算
        • 引入包：<script src="https://cdn.bootcdn.net/ajax/libs/decimal.js/10.2.0/decimal.min.js"></script>
        • new Decimal(a).add(new Decimal(b)).toNumber()加法
    • Null undefined
  2. 引用数据类型（引用类型）
    • Object 对象，内置对象 Function、Array、Date、RegExp、Error 等都是属于 Object 类型。也就是说，除了那七种基本数据类型之外，其他的，都称之为 Object 类型。

      函数的返回值

  • 如果函数中不写return，则也会返回undefined

    函数名、函数体和函数加载问题

  • 函数的加载问题：JS加载的时候，只加载函数名，不加载函数体。所以如果想使用内部的成员变量，需要调用函数。
  • fn() 和 fn 的区别【重要】
    • fn()：调用函数。调用之后，还获取了函数的返回值。
    • fn：函数对象。相当于直接获取了整个函数对象

  • 事件的三要素

      • 绑定事件： 事件源box.事件onclick = function(){事件驱动程序};

      DOM 简介和DOM操作

      简介

    • 文档对象模型（Document object Model），操作网页上的元素的API。比如让盒子移动、变色、轮播图等。
    • 节点
      • 文档节点（文档）：整个 HTML 文档。整个 HTML 文档就是一个文档节点。
      • 元素节点（标签）：HTML标签。
      • 属性节点（属性）：元素的属性。
      • 文本节点（文本）：HTML标签中的文本内容（包括标签之间的空格、换行）。

        获取节点

  • 元素节点的获取
    • var div1 = document.getElementById("box1"); //方式一：通过 id 获取 一个 元素节点（为什么是一个呢？因为 id 是唯一的）
  • DOM访问关系的获取
    • 父节点节点.parentNode
    • 获取兄弟节点
      • 下一个节点nextSibling
      • 下一个元素节点nextElementSibling
      • 上一个节点previousSibling
      • 下一个元素节点previousElementSibling
    • 获取子节点
      • firstChild firstElementChild
      • lastChild lastElementChild
      • childNodes children
