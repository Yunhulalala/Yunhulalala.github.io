Git

发表于2022-03-22更新于2022-03-30

创建本地仓库

git init

工作区状态

git status

查看历史提交记录

git log
git log --pretty=oneline

查看每次提交命令

git reflog

版本回退

git reset --hard <commit_id>

添加修改到暂存区

git add <filename>

查看工作区与版本库里面最新版本的区别

git diff HEAD -- <filename>

将暂存区的修改回退到工作区

git reset HEAD <filename>

丢弃工作区的修改

git checkout -- <filename>

添加到暂存区，并想要丢弃修改

  1. git reset HEAD <filename>
  2. git checkout -- <filename>

    撤销版本库提交

    git reset --hard <commit_id>

    删除版本库文件

  3. git rm <filename>
  4. git commit -m <info>

    远程库

    添加远程库

  5. 在Github创建一个新的仓库
  6. 创建本地仓库git init
  7. 关联仓库 git remote add origin <http|ssh>
  8. 推送master分支内容
  9. 连接不上远程仓库解决方法
    git config –global http.proxy http://127.0.0.1:portnum
    git config –global https.proxy http://127.0.0.1:portnum
