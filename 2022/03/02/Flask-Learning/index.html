第三章 模板

  • 把HTML代码存储在单独的文件中
  • 借助模板引擎，我们可以在HTML文件中使用特殊的语法来标记出变量，这类包含固定内容和动态部分的可重用文件称为模板（template）。

模板基本用法

模板语法

  • {%...%}if判断、for循环
  • {{...}}表达式
  • {#...#}注释

  • 全局对象

    • 使用

  • 模板环境对象

    • 所有的配置选项、上下文变量、全局函数、过滤器和测试器都存储在Environment实例上

第四章 表单

  • 使用Flask-WTF处理表单
    • validators=[DataRequired()验证器
  • 在模板中渲染表单
    • {{ form.csrf_token }} <!-- 渲染CSRF令牌隐藏字段 -->
  • 处理表单数据
    • 验证表单数据
      • 客户端服务端验证
        • 客户端验证
          • 使用HTML5内置的验证属性即可实现基本的客户端验证（type、required、min、max、accept等
        • 服务器端验证
      • WTForms验证机制
        • 对表单实例调用validate（）方法。这会逐个对字段调用字段实例化时定义的验证器，返回表示验证结果的布尔值。如果验证失败，就把错误消息存储到表单实例的errors属性对应的字典中
      • 在视图函数中验证表单
        • if request.method == 'POST' and form.validate(): # 处理POST请求
        • if form.validate_on_submit():
  • 使用宏渲染表单
  • 自定义验证器
  • 文件上传
